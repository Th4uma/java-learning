**实验一、Java语言基础**

 

 

一、实验目的

1、搭建Java平台

1、掌握Java语言的基本数据类型

2、掌握Java数组

3、掌握Java运算符和表达式

 

二、实验内容

（1）编写一个程序，输出Unicode字符表中从字符’1’开始的20个字符。

（2）编写一个程序，用户从键盘输入商品的单价和数量，程序输出总价。

（3）数组的基本知识：

  ①声明并创建一个有3个double型元素的数组A，有5个double型元素的数组B，有1个字符型元素的数组C，以及有2个逻辑型元素的数组D；

  ②分别输出这四个数组的引用及第一个元素的值。

  ③将A中3个元素初始化为2.5、3.6、5.8，将B中5个元素初始化为100.5，56，98，43.2，71.6；

  ④输出数组A与数组B；

  ⑤使用赋值运算符将B赋给A；

  ⑥将B中第二个元素的值改为4；

  ⑦再次输出A、B的引用

  ⑧输出A,B的元素值

  ⑨运行代码，对每条输出语句，用一句话解释该输出结果产生的原因及内存中的变化。

（4）假设x=24,y=156

①将x左移2位，并赋值给x，分别输出左移前后x的值。

②将y右移1位，并赋值给y，分别输出右移前后y的值。

③对x,y分别进行按位与、按位或、按位异或运算，输出运算结果。

④使用条件运算符计算x，y中的较小值，并输出。

（5）对本次上机进行小结，简单总结本次上机练习遇到的问题，解决的办法等。

三、要求

（1）代码必须放在以自己名字和学号命名的文件夹中，并在控制台上进行编译和运行；

（2）源代码与运行结果都必须上传。
