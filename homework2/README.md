**
** **实验二、Java语句**

 

 

一、实验目的

1、掌握Java的分支语句、循环语句、跳转语句，并熟练运用到求解实际问题中

 

二、实验内容

（1）用条件分支语句实现以下内容：给定一个人的年龄，输出这个人是少年（0到19岁），青年（20岁到39岁），中年（40岁到59岁）还是老年（60岁以上）。

（2）用switch语句实现以下内容：根据预先给定的玩家A,B,C,D的四个等级，分别输出相应的评语：

  A：非常棒！

  B：很好！

  C：您通过了

  D：下次努力

最后输出：您的等级是XX。

如：若玩家的等级为B，则输出：很好！您的等级是B。

（3）采用选择排序方法实现数组{1,4,6,2,8,3,5,9,7}从大到小的排列。（选择排序法：从1到n，选出值最大的元素，交换到第一个位置；然后从2到n，选择除值最大的元素，交换到第二个位置…依此类推）。

（4）使用while循环与跳转语句获取1~100中奇数之和（即1+3+5+…+99的值），并打印。

（5）对本次上机进行小结，简单总结本次上机练习遇到的问题，解决的办法等。

 

三、要求

（1）代码必须放在以自己名字和学号命名的文件夹中，并在控制台上进行编译和运行；

（2）源代码与运行结果都必须上传。
